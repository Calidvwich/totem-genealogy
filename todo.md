# 任务清单与完成状态

本文件记录当前 TotemDB 族谱系统需求的实现情况。

## 1. 环境与数据库

- [x] WSL `Ubuntu-18.04` 启动与连接流程。
- [x] TotemDB 使用 `55432` 端口启动。
- [x] 默认数据库 `genealogy`。
- [x] `init_db.sql` 初始化表结构。
- [x] `users`、`genealogies`、`collaborations`、`members`、`marriages`、`member_photos` 表。
- [x] 兼容 Totem 不支持部分 PostgreSQL 高级语法的情况。
- [x] 一键启动脚本 `start-genealogy.bat`。

## 2. 用户与权限

- [x] 用户登录与登出。
- [x] 普通用户公开注册，注册成功后回到登录页。
- [x] 密码哈希存储，禁止明文密码。
- [x] 管理员查看用户列表。
- [x] 管理员查看用户详情。
- [x] 管理员新增、修改、删除用户。
- [x] 公开注册禁止创建 `admin`。
- [x] 普通用户不能查看用户管理界面。
- [x] 族谱创建者默认是当前登录用户。
- [x] 创建者可授权、撤销协作者。

## 3. 族谱与成员管理

- [x] 族谱新建、修改、删除。
- [x] 成员新建、修改、删除。
- [x] 成员详情页。
- [x] 默认头像展示。
- [x] 上传照片并以 sha256 存入数据库。
- [x] 父母关系按姓名输入，重名时要求 ID 确认。
- [x] 子女关系按姓名输入，重名时要求 ID 确认。
- [x] 父亲必须是男，母亲必须是女。
- [x] 父母与子女必须在同一族谱。
- [x] 出生、结婚、离婚、死亡年份约束在数据生成和业务校验中处理。
- [x] 父母共同拥有子女时自动生成婚姻关系。
- [x] 移除共同子女关系后，自动移除无共同子女的婚姻关系。

- [x] 成员模糊搜索。

- [x] 搜索失败时红色提示。

- [x] 树形图缩放与重置。
- [x] 成员亲缘关系路径查询。
- [x] 查询配偶和子女。
- [x] 查询祖先。
- [x] 查询每代平均寿命。
- [x] 查询 50 岁以上男性单身成员。
- [x] 查询早于本代平均出生年份的成员。
